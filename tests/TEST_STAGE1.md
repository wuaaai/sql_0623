# 阶段1 验收测试

测试目标：验证"单表查询"三项核心能力
- 连接数据库 → 列出表 → 单表查询

## 连接状态

先确认左下角显示"已连接"（达梦 localhost:5236, schema=RDYS_PUBLIC_TBS），页面上能看到表列表。

---

## 测试1: 列出所有表

**输入:**
```
数据库中有哪些表
```

**验收标准:**
- LLM 调用了 list_tables 工具（不是瞎编的）
- 返回的表名与侧边栏一致（真实表名）
- 没有编造中文含义或分类

---

## 测试2: 查看单张表的数据

**输入:**
```
查看 RDYS_BAS_EXPFUNC 表的前5行数据
```

**验收标准:**
- LLM 调用了 run_sql
- SQL 使用了 ROWNUM <= 5（达梦语法，不是 LIMIT）
- 结果以表格形式展示
- Markdown 渲染正常（不是原始文本）

---

## 测试3: 统计表的行数

**输入:**
```
RDYS_BAS_MOFDIV 这张表有多少条数据
```

**验收标准:**
- LLM 生成 SELECT COUNT(*) FROM RDYS_BAS_MOFDIV
- 返回准确的行数
- 用自然语言向用户说明结果

---

## 通过条件

三项测试全部通过即阶段1验收完成。
