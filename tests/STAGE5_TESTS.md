# 阶段5 验收测试 — 深度洞察

> 子查询/占比/环比/异常检测。25题，全部口语化。

## 一、子查询（5题）

| # | 用户问题 | 预期行为 |
|---|---------|---------|
| 5.1 | 哪些地市一般公共预算收入超过了全省平均水平 | 先算AVG → WHERE BYS_JE > 平均值 |
| 5.2 | 一般公共预算支出低于全省平均水平的地市有哪些 | WHERE BYS_JE < 平均值 |
| 5.3 | 比石家庄一般公共预算收入高的地市有哪些 | WHERE BYS_JE > (SELECT BYS_JE WHERE RG_NAME='石家庄市') |
| 5.4 | 一般公共预算收入超过全省平均水平两倍的地市 | WHERE BYS_JE > AVG * 2 |
| 5.5 | 各地市一般公共预算收入和全省最高收入的差距有多大 | 各市值 - MAX值 |

## 二、占比分析（5题）

| # | 用户问题 | 预期行为 |
|---|---------|---------|
| 5.6 | 各地市一般公共预算收入占全省的比例分别是多少 | 各市/全省合计 ×100 |
| 5.7 | 税收收入占一般公共预算收入的比重是多少 | 税收行/合计行 |
| 5.8 | 收入前3名的地市合计占了全省多大比例 | TOP3合计/全省合计 |
| 5.9 | 各地市社保基金支出占全省社保支出的比例 | 各市/全省合计 |
| 5.10 | 政府性基金收入前5名地市合计占总体的比重 | TOP5合计/全省合计 |

## 三、环比分析（4题）

| # | 用户问题 | 预期行为 |
|---|---------|---------|
| 5.11 | 这个月一般公共预算收入和上个月比涨了多少跌了多少 | LAG窗口函数取上月值→计算差值 |
| 5.12 | 今年各月一般公共预算收入环比变化趋势 | 逐月LAG→环比增长率 |
| 5.13 | 哪些地市本月一般公共预算收入环比下降了 | LAG→WHERE 本月<上月 |
| 5.14 | 最近半年社保基金收入环比下降的月份有哪些 | LAG→筛选环比负值 |

## 四、异常检测（4题）

| # | 用户问题 | 预期行为 |
|---|---------|---------|
| 5.15 | 一般公共预算收入有没有特别异常的地市 | 均值±2倍标准差标记异常 |
| 5.16 | 各地市一般公共预算收入同比波动最大的是哪些 | 计算同比率方差→排序 |
| 5.17 | 今年有没有一般公共预算收入突然大幅下降的月份 | LAG对比→降幅>阈值标记 |
| 5.18 | 一般公共预算支出中，有没有偏离平均水平特别大的项目 | 均值±N标准差检测 |

## 五、组合场景（4题）

| # | 用户问题 | 预期行为 |
|---|---------|---------|
| 5.19 | 各地市占比排名，有没有一市独大的 | 占比+排名+判断集中度 |
| 5.20 | 本月环比变化最大的地市，和去年同期对比 | 环比+同比组合分析 |
| 5.21 | 收入异常高的地市，支出占比是不是也异常 | 分别异常检测→关联分析 |
| 5.22 | 环比下降的地市里面，哪些连续两个月都在降 | 连续两月LAG→双重筛选 |

## 六、边界（3题）

| # | 用户问题 | 预期行为 |
|---|---------|---------|
| 5.23 | 这个月刚出来的数据和上个月比有什么变化 | 最新两月数据LAG对比 |
| 5.24 | 所有地市收入都一样，怎么分析异常 | 标准差为0→告知均匀分布 |
| 5.25 | 如果全部都标记为异常了是怎么回事 | 检查阈值合理性→建议调整 |

## 验收标准

- P0: 子查询+占比 10题 ≥80%正确
- P0: 环比 4题 ≥75%正确使用LAG
- P1: 异常检测 4题 阈值合理
- P1: 组合场景 4题 综合运用
- P2: 边界 3题 降级建议合理
