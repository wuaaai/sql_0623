# 全局记忆 (L2)

## 达梦语法要点
- 限制行数: WHERE ROWNUM <= N (不是 LIMIT)
- 字符串拼接: 使用 || (不是 CONCAT)
- ORDER BY + ROWNUM: 必须嵌套子查询 (ROWNUM先于ORDER BY执行)

## 数据库概况
- Schema: RDYS_PUBLIC_TBS，共47张表

## 四大预算类型 → 表名前缀映射

| 用户口语 | 表名前缀 | 数量 |
|----------|----------|------|
| 一般公共预算 | YBGGYS | 7张 |
| 社会保险/社保基金 | SHBXJJ | 7张 |
| 国有资本/国资预算 | GYZBJY | 7张 |
| 政府性基金 | ZFXJJ | 4张 |

## 预算表通用字段

| 字段 | 含义 |
|------|------|
| RG_CODE / RG_NAME | 地区编码/名称 |
| YEAR_MONTH | 年月 (格式: 202001) |
| XM_CODE / XM_NAME | 项目编码/名称 |
| YSS | 预算数 |
| BYS_JE / BY_JE | 本月金额 |
| BYS_SNTYS / BY_SNTYS | 上年同期金额 |
| BYS_TBE / BY_TBE | 同比增减额 |
| BYS_TBB / BY_TBBFS | 同比增减率(%) |
| BYLJS_JE | 本月累计金额 |

**列名变体**: BYS_JE vs BY_JE, BYS_SNTYS vs BY_SNTYS, BYS_TBB vs BY_TBBFS — 用 describe_table 确认具体表的列名。

## 地区编码
- 130000000: 河北省本级
- RG_NAME LIKE '%本级%': 省级数据
- RG_NAME NOT LIKE '%本级%': 各地市

## 项目名称常见值
- "合    计" / "合计" / "一、收入合计"
- "一、税收收入" / "二、非税收入"

## 排名查询（重要）
- 一般公共预算有专门排名表: RDYS_LD_YSZX_YBGGYS_SZPMQK
  - SR_LJWCS_SORT: 收入排名 | SR_TBE_SORT: 收入同比排名
  - ZC_LJWCS_SORT: 支出排名 | ZC_TBE_SORT: 支出同比排名
  - 直接 SELECT * ORDER BY 对应排序列即可，不需要 GROUP BY
- 其他预算类型无排名表，用 ORDER BY + ROWNUM
