# 数据库探索技能

## 何时使用
用户想了解数据库结构、查看表、了解列信息时触发。

## 操作步骤

1. `list_tables` — 获取所有表名
2. `run_sql` — 用 `SELECT * FROM table_name WHERE ROWNUM <= 1` 查看表结构
3. `run_sql` — 用 `SELECT COUNT(*) FROM table_name` 了解数据量

## 规则

- **仅列出真实表名**，不编造中文含义或分类
- 表名是数据库中的原始名称，不要重命名
- 用表格形式列出表名
- 如果表超过30张，分组展示，每组不超过15张
- 禁止将表名翻译或推测其业务含义
