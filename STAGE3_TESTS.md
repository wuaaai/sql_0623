# 阶段3 验收测试 — 聚合分析

> 所有问题均为预算分析人员真实口语。Agent 自动判断用预聚合列查还是需要 GROUP BY。
> 实测发现数据库已有大量预聚合列（累计值、同比率、排名表），多数统计问题无需 GROUP BY。

---

## 一、排名查询

### 收入排名

| # | 用户问题 | 预期行为 |
|---|---------|---------|
| 3.1 | 一般公共预算收入最高的5个地市 | 查排名表 SZPMQK 的 SR_LJWCS_SORT 列，或用 run_sql ORDER BY 收入列 DESC + ROWNUM |
| 3.2 | 一般公共预算收入最低的3个地市 | ORDER BY 收入列 ASC + ROWNUM <= 3 |
| 3.3 | 社保基金收入排名前十的地区 | 查社保表 LJ_JE 列 ORDER BY DESC + ROWNUM <= 10 |
| 3.4 | 国有资本预算收入哪个地市最高 | ORDER BY 收入列 DESC + ROWNUM <= 1 |
| 3.5 | 政府性基金收入排名，从低到高 | ORDER BY 收入列 ASC |
| 3.6 | 各市一般公共预算收入排名 | 排除省本级，按收入降序排列所有地市 |
| 3.7 | 一般公共预算支出排名前5 | 查支出表或 SZPMQK 的 ZC_LJWCS_SORT |

### 同比排名

| # | 用户问题 | 预期行为 |
|---|---------|---------|
| 3.8 | 一般公共预算收入同比增长最快的5个地区 | ORDER BY BYS_TBB DESC 或查 SZPMQK 的 SR_TBE_SORT |
| 3.9 | 社保基金收入同比降幅最大的地区 | ORDER BY BYS_TBB ASC（取负值最大的） |
| 3.10 | 哪些地市一般公共预算收入同比下降了 | WHERE BYS_TBE < 0 或 SR_TBE < 0 |
| 3.11 | 国有资本预算收入同比增长超过20%的地区 | WHERE BYS_TBB > 20 |
| 3.12 | 政府性基金支出同比降幅排名前5 | ORDER BY 支出同比列 ASC + ROWNUM <= 5 |

### 累计排名

| # | 用户问题 | 预期行为 |
|---|---------|---------|
| 3.13 | 今年社保基金累计收入排名 | 查 LJ_JE（累计收入）列 ORDER BY DESC |
| 3.14 | 一般公共预算累计支出最高的地市 | 查 BYLJS_JE 或 ZC_LJWCS 列 ORDER BY DESC + ROWNUM <= 1 |

## 二、汇总统计

| # | 用户问题 | 预期行为 |
|---|---------|---------|
| 3.15 | 2025年全年一般公共预算收入总共是多少 | 查12月累计列 BNSY_LJS 或 BYLJS_JE，或 SUM(BYS_JE) WHERE YEAR_MONTH LIKE '2025%' |
| 3.16 | 今年上半年社保基金支出合计 | WHERE YEAR_MONTH BETWEEN '202601' AND '202606'，SUM 支出列 |
| 3.17 | 2025年政府性基金收入汇总 | 查累计列或 SUM |
| 3.18 | 全省一般公共预算收入总共有多少 | 查合计行（XM_NAME LIKE '%合%计%'），不需要 GROUP BY |
| 3.19 | 今年国有资本预算累计支出多少 | 查 BYLJS_JE 或 LJ_JE 列 |
| 3.20 | 2025年第一季度（1-3月）一般公共预算收入 | WHERE YEAR_MONTH IN ('202501','202502','202503')，SUM 或查3月累计列 |

## 三、时间趋势

| # | 用户问题 | 预期行为 |
|---|---------|---------|
| 3.21 | 今年各月一般公共预算收入变化趋势 | SELECT YEAR_MONTH, BYS_JE FROM table WHERE YEAR_MONTH LIKE '2026%' AND XM_NAME LIKE '%合%计%' ORDER BY YEAR_MONTH |
| 3.22 | 2025年各月社保基金支出走势 | ORDER BY YEAR_MONTH ASC，WHERE YEAR_MONTH LIKE '2025%' |
| 3.23 | 最近半年的一般公共预算收入逐月变化 | resolve_time("最近6个月") → YEAR_MONTH IN (...) ORDER BY YEAR_MONTH |
| 3.24 | 政府性基金各月收入波动情况 | 逐月 ORDER BY，标注最高最低月份 |
| 3.25 | 今年各月一般公共预算收入，和去年同期对比 | 选 BYS_JE（本月）和 BYS_SNTYS（上年同期）两列，逐月展示 |
| 3.26 | 国有资本预算上半年各月支出趋势 | WHERE YEAR_MONTH BETWEEN '202601' AND '202606' ORDER BY YEAR_MONTH |

## 四、统计指标（平均/最高/最低）

| # | 用户问题 | 预期行为 |
|---|---------|---------|

## 五、多维分组（真正需要 GROUP BY 的场景）

这些场景需要跨行聚合，因为预聚合列不够用。

| # | 用户问题 | 预期行为 |
|---|---------|---------|
| 3.31 | 各市2025年全年一般公共预算收入汇总 | 需要 GROUP BY RG_NAME + SUM(BYS_JE) WHERE YEAR_MONTH LIKE '2025%'，因为要跨12个月汇总每个市 |
| 3.32 | 各市各月的社保基金收入 | GROUP BY RG_NAME, YEAR_MONTH（多维分组） |
| 3.33 | 今年上半年各市一般公共预算收入排名 | GROUP BY RG_NAME, SUM(BYS_JE) WHERE YEAR_MONTH BETWEEN '202601' AND '202606' |
| 3.34 | 各市2025年一般公共预算税收收入和非税收入分别汇总 | GROUP BY RG_NAME, XM_NAME（按税收/非税分类汇总） |
| 3.35 | 各月四大预算类型的收入汇总对比 | 现阶段分表查询，分别展示四类预算的月度汇总 |

## 验收标准

| 级别 | 标准 |
|------|------|
| P0 | 排名/汇总/趋势三大类共20题，≥90%回答正确 |
| P0 | 回答内容基于真实数据，不编造数字 |
| P1 | 多维分组场景正确使用 GROUP BY |
| P1 | 结果展示包含排名序号、地区名、数值 |
| P2 | 跨预算类型对比能分别查询后给出对比结论 |
