# 阶段2 测试问题集 — 预算业务精准查询

> 设计原则：用户输入均为预算分析员真实口语表达，覆盖四大预算类型、收入支出双场景、每子功能 5-8 题含边界场景

## 2.2 按地区筛选

| # | 优先级 | 用户输入 | 预期 SQL |
|---|--------|---------|----------|

| 2.2.2 | P0 | 石家庄的情况呢 | `WHERE RG_NAME LIKE '%石家庄%'` |
| 2.2.3 | P0 | 省本级的数据 | `WHERE RG_NAME LIKE '%本级%'` |
| 2.2.4 | P0 | 看各地市的，不要省本级 | `WHERE RG_NAME NOT LIKE '%本级%'` |
| 2.2.5 | P1 | 石家庄和保定两个地方 | `WHERE RG_NAME LIKE '%石家庄%' OR RG_NAME LIKE '%保定%'` |

| 2.2.8 | P2 | 我想看开发区那些 | LIKE '%开发区%' 或 LIKE '%高新%' |

## 2.5 同比对比查询（核心高频场景）

| # | 优先级 | 用户输入 | 预期行为 |
|---|--------|---------|---------|
| 2.5.1 | P0 | 河北1月收入比去年涨了还是跌了 | 选 BYS_JE, BYS_SNTYS, BYS_TBE, BYS_TBB |
| 2.5.2 | P0 | 哪些地区收入同比下降了 | `WHERE BYS_TBE < 0` |
| 2.5.3 | P0 | 社保基金收入同比增长超过10%的地区 | `WHERE BYS_TBB > 10` |
| 2.5.4 | P0 | 和去年比，多了多少 | 选同比列 BYS_TBE + BYS_TBB |
| 2.5.5 | P1 | 国有资本预算收入和去年同期对比 | search_schema → GYZBJY 收入表 → 同比列 |
| 2.5.6 | P1 | 政府性基金支出同比降幅最大 | `WHERE BYS_TBE < 0 ORDER BY BYS_TBE ASC` |
| 2.5.7 | P2 | 这个月收入和上个月比 | 识别"环比"≠"同比"，提示差异 |
| 2.5.8 | P2 | 那些"收入为负增长"的 | 理解"负增长"=`WHERE BYS_TBB < 0` |
